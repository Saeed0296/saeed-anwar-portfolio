# Saeed Anwar - AI/ML Portfolio

Premium static portfolio foundation built with pure HTML, CSS, and vanilla JavaScript.

## Stack
- HTML5
- CSS3 (custom design system, glassmorphism, responsive utilities)
- Vanilla JavaScript

## Local Preview
Open `index.html` directly in a browser, or serve the folder with any static server.

## Structure
- `index.html` - main application shell and sections
- `assets/css/main.css` - reset, tokens, components, layout system, responsive rules
- `assets/js/main.js` - small progressive enhancement scripts

## Deployment
This project is static and can be hosted on GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
